# Final Submission Checklist

## Code

- [ ] `app.py` launches without errors.
- [ ] `requirements.txt` is complete.
- [ ] Model loads once and is reused.
- [ ] Inference works for custom image and question.
- [ ] Invalid inputs are handled gracefully.
- [ ] README has exact setup and run commands.

## Dataset

- [ ] At least 30 image-question-answer pairs.
- [ ] At least 5 question types; target is 6.
- [ ] Every image path is valid.
- [ ] Ground-truth answers are reviewed.
- [ ] Easy and challenging examples are included.

## Evaluation

- [ ] Overall accuracy calculated.
- [ ] Per-question-type accuracy calculated.
- [ ] Average inference time calculated.
- [ ] Failure cases saved and explained.
- [ ] Charts exported for the report.

## Report and Demo

- [ ] 3 to 5 page report completed.
- [ ] Architecture diagram included.
- [ ] Screenshots included.
- [ ] 2-minute demo video recorded.
- [ ] Fresh-clone test completed.
- [ ] Backup screenshots ready.
